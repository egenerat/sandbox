# C++

Install and check version

```
sudo apt install build-essential
gcc -v
```

Build and run
```
g++ hello.cpp
./a.out
```

Threads
```
g++ threads.cpp -lpthread
```

Boost missing (Ubuntu)
```
sudo apt install libboost-all-dev
```
